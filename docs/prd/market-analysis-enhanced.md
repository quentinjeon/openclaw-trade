# PRD: 시황 분석 고도화

> **작성일**: 2026-03-09  
> **버전**: 1.0.0  
> **상태**: 승인

---

## 1. 개요

현재 시황 분석 페이지는 카드 형태의 지표 표시에 그치고 있다.  
업비트/바이낸스 수준의 **트레이딩뷰 캔들 차트 + 실시간 시세 + 코인 워치리스트**를 갖춘 전문 트레이더 화면으로 고도화한다.

## 4. 기능 명세

### 4.1 헤더 영역
- 심볼 선택 드롭다운 (BTC/USDT, ETH/USDT)
- 현재가 (크게 표시, 실시간 갱신)
- 24h 변동률 (색상: 상승=초록, 하락=빨강)
- 24h 고가 / 저가 / 거래량(USDT)

### 4.2 타임프레임 선택
| 버튼 | API 파라미터 | 설명 |
|------|------------|------|
| 1분 | `1m` | 단기 스캘핑 |
| 5분 | `5m` | 단기 |
| 15분 | `15m` | 단기~중기 |
| 1시간 | `1h` | 중기 (기본값) |
| 4시간 | `4h` | 중기~장기 |
| 1일 | `1d` | 장기 |

### 4.3 캔들 차트 (TradingView Lightweight Charts)
- 오픈소스 `lightweight-charts` 라이브러리 사용
- 캔들스틱 차트 (상승: 파랑, 하락: 빨강 — 한국식)
- 오버레이 지표:
  - MA 20 (노랑)
  - MA 50 (주황)
  - MA 200 (보라)
  - 볼린저밴드 상/하단 (회색 점선)
- 하단 서브차트: 거래량 바
- 크로스헤어: 마우스 호버 시 OHLCV 표시
- 자동 스케일 + 드래그/줌 지원

### 4.4 오른쪽 워치리스트
- 기본 코인: BTC, ETH, SOL, BNB, XRP (USDT 페어)
- 각 코인: 심볼, 현재가, 24h 변동률
- 클릭 시 해당 코인 차트로 전환
- 활성 코인 하이라이트

### 4.5 오른쪽 지표 패널
- 방향성: BULLISH / BEARISH / NEUTRAL 배지
- RSI 값 + 상태 (과매수/과매도/중립)
- MACD Histogram 방향 (양/음)
- 볼린저밴드 위치 (상단/중간/하단)
- OpenClaw 매매 신호

  "candles": [

  ]

    { "symbol": "ETH/USDT", "price": 1941.66, "change_24h": -1.72 },

  ]
}

## 6. 기술 스택

| 항목 | 기술 |
