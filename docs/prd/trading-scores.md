# PRD: 매수·매도·보유 점수 (요약)

## 목적
심볼별 **매수 매력 / 매도 압력 / 보유 유지** 점수(0~100)로 자동매매 방향·**매수 시 투입 비중**을 정하고, 대시보드에서 한눈에 확인.

## 점수 구성 (1h OHLCV)
- Williams %R (Larry 스타일 과매도·과매수)
- RSI(14), MACD 히스토그램, 종가 vs SMA20, 시장 분석 방향(BULLISH 가산)

## 매매 규칙 (요약)
- **무포지션**: 매수점수가 매도보다 충분히 크면 BUY, 전략 신호와 병합.
- **보유 중**: 매도점수가 보유유지보다 크면 SELL, 아니면 HOLD.
- **비중**: `alloc_mult ≈ 0.18 + 0.82×(매수점수/100)` × 리스크 매니저 기본 포지션%.

## API
- `GET /api/trading-scores/` — 심볼별 점수 + 포트폴리오 권장 투자/현금 비중 힌트.
