# PRD: 내 지갑 (My Wallet) 페이지

> **버전**: 1.0.0  
> **작성일**: 2026-03-09  
> **상태**: 개발 중

---

## 개요

Binance 계좌와 연동된 실시간 지갑 잔액을 조회하는 전용 페이지.  
페이퍼트레이딩/실거래 모드 모두 지원하며, 코인별 보유 수량, USD 및 KRW 환산 가치, 비중을 한눈에 확인할 수 있다.

---

## 사용자 스토리

- 트레이더로서, 내 Binance 지갑의 코인별 잔액을 한눈에 보고 싶다.
- 트레이더로서, 각 코인의 USD 및 KRW 환산 가치를 확인하고 싶다.
- 트레이더로서, 전체 지갑 총 가치와 코인별 구성 비율을 시각적으로 보고 싶다.
- 트레이더로서, 지갑 데이터가 자동으로 갱신되길 원한다.
- 트레이더로서, 페이퍼트레이딩 모드와 실거래 모드를 명확히 구분하고 싶다.
- 트레이더로서, 미소량(먼지) 자산을 숨기고 싶다.

## 기술 스펙

### 백엔드
- **파일**: `backend/routers/wallet.py`
- **등록**: `backend/main.py`에 라우터 추가
- **의존성**: 기존 `ExchangeConnector` 재사용 (`fetch_balance`, `fetch_ticker`)
- **환율**: `backend/routers/market.py`의 `_fx_cache` 재사용 (5분 캐시)
- **스택오버플로 방지**: USDT는 가격 조회 스킵, 직접 1.0 처리
- **소액 필터**: 0.0001 USD 미만 자산 선택적 제외

### 프론트엔드
- **페이지**: `frontend/src/app/wallet/page.tsx`
- **컴포넌트**: `frontend/src/components/wallet/`
  - `WalletSummaryCards.tsx` — 4개 요약 카드
  - `AssetTable.tsx` — 코인별 자산 테이블 (프로그레스 바 포함)
- **타입**: `frontend/src/types/wallet.ts`
- **API**: `frontend/src/services/api.ts`에 `walletApi` 추가
- **사이드바**: `Sidebar.tsx`에 "내 지갑" 메뉴 추가 (`Wallet` 아이콘)
- **갱신 주기**: SWR 30초 자동 갱신 + 수동 새로고침 버튼
- **소액 숨기기**: 토글 버튼으로 $0.01 미만 자산 숨김

## 비즈니스 로직

### 자산 가치 계산
1. `fetch_balance()` → 코인별 보유 수량
2. USDT는 1.0 USD 고정
3. 그 외 코인: `fetch_ticker("{CURRENCY}/USDT")` → 현재가 조회
4. `usd_value = amount × current_price_usd`
5. `krw_value = usd_value × fx_rate`
6. `pct_of_total = (usd_value / total_usd) × 100`

### 페이퍼트레이딩 모드
- `ExchangeConnector._paper_balance`에서 직접 읽음
- 실거래와 동일한 API 구조 반환
